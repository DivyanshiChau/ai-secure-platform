# AI Secure Data Intelligence Platform

## Overview
This project analyzes logs and detects sensitive data such as passwords, API keys, and emails.

## Features
- Log file analysis
- Sensitive data detection
- Risk scoring engine
- AI-ready architecture

## Tech Stack
- FastAPI (Backend)
- HTML/CSS/JS (Frontend)

## Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload

### Frontend
Open frontend/index.html in browser
